# Minimal [JsonRPC 2.0](http://www.jsonrpc.org/specification) api project (in Elixir)

## Depends on
 * {:poison, "~> 3.1"},
 * {:cowboy, "~> 1.1"},
 * {:jsonrpc2, "~> 1.0"},
 * {:plug, "~> 1.5"}

## How to run 
```
mix deps.get
mix run
```
